FONTLOG 
Unicode BMP Fallback SIL font
=================

This file provides detailed information on the Unicode BMP Fallback SIL font software. 
This information should be distributed along with the Unicode BMP Fallback SIL font 
and any derivative works.

Basic Font Information
----------------------

Intended for debugging, this font contains a glyph for every character in the 
Basic Multilingual plane (including Private Use Area) of Unicode, each glyph 
consisting of a box enclosing the four hex digits identifying the Unicode scalar value.

Current version supports BMP characters from Unicode 6.1.

One font from this typeface family is included in this release:

     * Unicode BMP Fallback SIL Regular
